# Backlink Building Strategy

## Quick Wins (Week 1)

### 1. Directory Submissions (30 minutes)
**High-authority business directories:**
- [ ] Google Business Profile (https://business.google.com) — **Priority 1**
- [ ] Bing Places (https://www.bingplaces.com)
- [ ] Clutch.co (B2B services directory)
- [ ] GoodFirms (IT services)
- [ ] Wellfound (AngelList) — startup/tech focus
- [ ] Crunchbase — company profile
- [ ] ProductHunt — launch each product separately

**Spain-specific:**
- [ ] Google Maps (Madrid, Spain location)
- [ ] Páginas Amarillas (https://www.paginasamarillas.es)

**Action:** Create business listings with NAP (Name, Address, Phone) consistency. Link to techconcepts.org in every profile.

### 2. Developer Communities (1 hour)
**Active participation = natural backlinks:**
- [ ] GitHub README — add link in your personal profile README
- [ ] Dev.to — cross-post blog content (canonical URL to techconcepts.org)
- [ ] Hashnode — same as Dev.to
- [ ] HackerNews — share blog posts (no self-promotion, genuine contributions only)
- [ ] Reddit r/webdev, r/SaaS — answer questions, link when genuinely helpful
- [ ] Indie Hackers — share build stories

**Action:** Cross-post all 6 blog posts to Dev.to and Hashnode with canonical URL back to techconcepts.org.

### 3. GitHub Projects (30 minutes)
**Leverage your open-source work:**
- [ ] MBOX Converter — create GitHub repo, link to blog post in README
- [ ] Claude Code Power Pack — GitHub repo with installation guide
- [ ] All other tools — if open-source, link to corresponding blog/product page

**Action:** Add "Built by TechConcepts | Read the story: [blog link]" to README footers.

## Medium-Term (Month 1)

**Target publications:**
- [ ] Smashing Magazine (web dev)
- [ ] CSS-Tricks (frontend)
- [ ] LogRocket Blog (SaaS/product)
- [ ] Medium publications (Better Programming, The Startup)
- [ ] FreeCodeCamp News

**Action:** Draft 3 guest post pitches using existing blog content. Include author bio with link to techconcepts.org.

### 5. Tool Listings (2 hours)
**Submit products to:**
- [ ] Product Hunt (launch each product separately, 1/week)
- [ ] AlternativeTo (compare to similar tools)
- [ ] Slant.co (software recommendations)
- [ ] G2 (if applicable)
- [ ] Capterra (business software)

**Action:** Create Product Hunt launch calendar for 9 products (1 per week = 9 weeks of launches).

### 6. Content Syndication (1 hour)
**Republish with canonical tags:**
- [ ] Medium (import blog posts, set canonical URL)
- [ ] LinkedIn Articles (reformat for LinkedIn audience)
- [ ] Substack (if you start newsletter)

**Action:** Set up Medium publication, import all 6 blog posts with canonical URLs.

## Measurement

### Track These Metrics (Weekly)
- **Backlink count:** Use Google Search Console "Links" report
- **Referring domains:** Aim for +5/month in first quarter
- **Domain authority:** Check via Moz/Ahrefs (target DA 30+ by Q2)
- **Referral traffic:** GA4 → Acquisition → Traffic acquisition

### Target Milestones
- **Week 1:** 10 backlinks (directories + GitHub)
- **Month 1:** 25 backlinks (+ guest posts)
- **Month 3:** 50 backlinks (+ podcast, partnerships)
- **Month 6:** 100 backlinks (+ resource pages)

```
Hi [Editor],

I'm Evgeny, founder of TechConcepts. I recently published "[Your Blog Post]" which got [X engagement].

I'd like to write a more detailed version for [Publication]. Proposed title: "[Expanded Title]"

Outline:
1. [Section 1]
2. [Section 2]
3. [Section 3]

The piece would be 1,500-2,000 words with code samples and real-world examples.

Previously at Yandex (Search) and EY (Advisory). Darden MBA.

Available to write this in the next 2 weeks. Let me know if this fits your editorial calendar.

Best,
Evgeny
https://techconcepts.org
```

### Broken Link Outreach
```
Subject: Broken link on [Page Title]

Hi [Name],

I was reading your article "[Article Title]" and noticed the link to [broken URL] returns a 404.

I have a similar resource that might work as a replacement: [your URL]

It covers [brief description].

Happy to help if you'd like to update the link.

Best,
Evgeny
```
