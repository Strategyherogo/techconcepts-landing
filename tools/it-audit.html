IT Audit Checklist

25 critical checkpoints across security, compliance, infrastructure, operations, and documentation. Free, no signup required.

Checklist Progress 0 of 25 completed
Security (5 items)
Multi-factor authentication (MFA) enabled for all admin accounts
Password policy enforces minimum 12 characters with complexity requirements
Regular security patch schedule documented and followed
Access reviews conducted quarterly (who has access to what)
Offboarding checklist includes revoking all system access within 24 hours
Compliance (5 items)
Data retention policy documented and automated where possible
Privacy policy up to date with GDPR, CCPA, or relevant regulations
Audit logs enabled for critical systems and retained for at least 90 days
Vendor risk assessment process in place for third-party tools
Data processing agreements (DPAs) signed with all vendors handling personal data
Infrastructure (5 items)
Automated backups tested and verified monthly
Disaster recovery plan documented with RTO and RPO targets
Infrastructure as code (IaC) used for provisioning and configuration
Monitoring and alerting in place for critical systems (uptime, errors, performance)
Cloud cost optimization reviewed quarterly (unused resources, rightsizing)
Operations (5 items)
SaaS license audit conducted at least twice per year (unused seats)
IT ticketing system in place with SLA tracking
Change management process for production deployments
Onboarding checklist automates provisioning of accounts and access
IT asset inventory maintained and up to date (hardware, software, licenses)
Documentation (5 items)
Network diagram documenting all systems and data flows
Runbooks for critical incidents (outages, security events)
API documentation kept up to date with code changes
Business continuity plan reviewed annually
Single source of truth for internal documentation (not scattered across Slack, email, wikis)
