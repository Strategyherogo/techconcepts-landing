# TechConcepts.org

Landing page and website deployment for techconcepts.org.

## Overview

This project contains the deployment configuration and analytics integration for the TechConcepts landing page.

## Tech Stack

- Static HTML/CSS landing page
- Google Analytics integration
- GitHub Pages deployment

## Repository

- GitHub: https://github.com/Strategyherogo/techconcepts-landing

## Setup

```bash
npm install
```

## Deployment

The site is deployed to techconcepts.org via GitHub Pages.

## Analytics

Uses Google Analytics Admin and Data APIs for analytics integration.
